## 注意事项

```python
#1、[1:4]，头闭尾开
#2、List中，被''或者""的元素就会打印''，而在字符串中不会，而且带着[]被一起打印出来
#3、*代表重复操作，+是连接运算符可以将两个变量连接输出
#4、for循环的迭代目标不能是一个数字，要么是列表、元组、字典、字符串或者是range(数字)
#5、字典的键必须是不可变类型，值无所谓
#6、list为空时，索引无效
```

## 基本语法

```python
#1、数字(不可变，一旦数值改变就会被赋予新的内存地址，可以通过del删除再重新赋值
)
num_name = 3
num_name = 4 #此时print(id(num))变化
#2、字符串(每个单词是它的一个元素)（不可变）
str_name = 'string'
#3、列表(每项是它的一个元素)（可变）
list_name = [1, 2, 3, 4]
list_name = [1, '3', 4, 6]
list_name = [1, 'hhh', 3]
#4、元组(不能二次赋值，不能更新)（不可变，它属于一变报错那种）
tuple_name = (1,3,5,1)
#5、字典作为键值对存在，通过key找value，不通过偏移存取（可变）
dict_name = {} #必须先创建字典才能赋键值对
dict_name[a] = 'b'
dict_name = {2: 'a', 'b': 4, 'c': 7}
print(dict_name) #输出完整字典
print(dict_name.keys()) #输出所有键
print(dict_name.values()) #输出所有值

###所有不可变的包括元组可以用del删除

#6、运算符
a / /b #整除，向下取整
a ** b #a的b次方
```
